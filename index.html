Método de Holliday-Segar

¿Cómo se calcula?

  • De 0kg a 10kg, se calcula 100cc por cada kilo.
  • Se suman 50cc por cada kilo de peso por arriba de 10kg, hasta 20kg.
  • De 20kg para arriba, se suman 20cc por cada kilo adicional.

Método de Superficie Corporal

¿Cómo se calcula?

  • Cuando el niño pesa mas de 30kg, se utiliza un cálculo diferente: Superficie corporal = ( (peso * 4) + 7) / (peso + 90)
  • Este resultado se multiplica por 1500 o por 2000 para hallar el valor del volumen diario en cc, y el médico decide cual de los dos resultados utilizar.
